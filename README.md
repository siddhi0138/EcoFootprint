## üöÄ Getting Started

### Prerequisites
- **Node.js** (version 16 or higher)
- **npm** or **yarn** package manager

> üí° **Tip**: Use [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) to easily manage Node.js versions

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/siddhi0138/EcoFootprint.git
   ```

2. **Navigate to project directory**
   ```bash
   cd EcoFootprint
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   - Navigate to `http://localhost:3000` to view the application
   - The development server includes hot-reloading for instant preview of changes
